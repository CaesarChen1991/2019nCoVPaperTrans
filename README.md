# 新型冠状病毒（2019-nCoV）相关论文非官方、非专业翻译

为了了解当前疫情的真实情况，我开始阅读相关的论文，但是最新的论文找不到中文版本，于是自己动手开始翻译，一点小成果希望与大家分享，让身处焦虑的朋友能多有一些一手资料可以参考。

这些论文国内媒体已经有一些转载和报道，本页面向需要具体了解论文全貌又不想花时间读原文的读者，限于本人英文及医学知识水平，翻译质量有限，不适用于医学专业读者，敬请原谅。
